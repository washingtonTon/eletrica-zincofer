# Como Manter

## CSS
- Adicione estilos em `src/css/components.css`
- Use variáveis CSS de `src/css/variables.css`

## JavaScript
- Novos módulos em `src/js/`
- Importe em `main.js`

## Imagens
- Coloque em `src/assets/images/`

## Deploy
```
npm install
npm run build (futuro)
```

## Padronização
- Tailwind + CSS custom
- Mobile-first
- JS modular
