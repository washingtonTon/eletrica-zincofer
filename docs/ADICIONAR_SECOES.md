# Adicionar Novas Seções

1. Crie a seção no `src/index.html`:
```html
<section id="nova-secao">
  <h2>Nova Seção</h2>
  <div>Conteúdo</div>
</section>
```

2. Adicione link no menu:
```html
<a href="#nova-secao">Nova Seção</a>
```

3. Estilize em `src/css/components.css`

4. Adicione JS se necessário em novo módulo
